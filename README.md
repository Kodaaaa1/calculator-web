# 🧮 Calculadora Web

Uma calculadora simples e funcional desenvolvida com **HTML, CSS e JavaScript**, com foco em prática de lógica e desenvolvimento front-end.

---

## 🚀 Acesse o projeto online
👉 https://kodaaaa1.github.io/calculator-web/

---

## 🎯 Funcionalidades

✅ Operações básicas:
- Soma (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)

✅ Interface simples e intuitiva  
✅ Design responsivo e moderno  
✅ Atualização dinâmica do visor  
✅ Tratamento de erros (exibe "Erro" em contas inválidas)

---

## 🛠️ Tecnologias utilizadas

- 🧱 HTML5
- 🎨 CSS3
- ⚙️ JavaScript
